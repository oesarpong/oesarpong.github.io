2A. AD0_VAL Definition

AD0 is a Analog-Data-Converter used as a data input pin. This value can be set to either 0 or 1 which will interface with either I2C or SPI depending on which address and data rate the user prefers. Setting the value to 0 indicates the default address and 1 would be used for an alternative.
